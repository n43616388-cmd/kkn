# Run Project

## Step 1: Install Node.js
Download:
https://nodejs.org/

## Step 2: Open Project Folder

Terminal me:

npm install

## Step 3: Start Project

npm run dev

Open:

http://localhost:5173


# Deploy on Netlify

1. Create Netlify account
2. Upload project to GitHub
3. Netlify → Add New Site
4. Select GitHub repository

Build Command:

npm run build

Publish Directory:

dist

Click Deploy
